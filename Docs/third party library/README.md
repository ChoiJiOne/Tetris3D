# Third Party Library (서드 파티 라이브러리)

이 문서는 `Tetris3D`의 *v0.0* 버전에서 사용하는 서드 파티 라이브러리에 대한 문서입니다.
<br><br>


## Third Party

이 프로젝트에서 사용하는 서드 파티 라이브러리 목록은 다음과 같습니다.
- [SDL2](https://github.com/libsdl-org/SDL)
- [json](https://github.com/nlohmann/json)
- [miniaudio](https://miniaud.io/)
- [stb](https://github.com/nothings/stb)
<br><br>


## [SDL2](https://github.com/libsdl-org/SDL)

SDL(Simple DirectMedia Layer)은 C언어로 짜여진 크로스플랫폼 멀티미디어 라이브러리입니다. 이 프로젝트에서는 윈도우와 이벤트 처리를 수행하기 위해서 사용되었습니다.
<br><br>


## [json](https://github.com/nlohmann/json)

json은 C++기반의 json 파일을 로딩 및 파싱하는 라이브러리입니다. 이 프로젝트에서는 json 파일을 로딩 및 파싱 처리를 수행하기 위해서 사용되었습니다.
<br><br>


## [miniaudio](https://miniaud.io/)

miniaudio는 오디오 재생 및 캡처 라이브러리입니다. 이 프로젝트에서는 사운드 로딩 및 출력 처리를 위해서 사용되었습니다.
<br><br>


## [stb](https://github.com/nothings/stb)

stb는 C/C++ 기반의 싱글 헤더 라이브러리 모음입니다. 이 프로젝트에서는 텍스처 파일, 트루 타입 폰트 파일 로딩, 폰트 텍스처 아틀라스, 텍스처 크기 변경 처리를 수행하기 위해서 사용되었습니다.
